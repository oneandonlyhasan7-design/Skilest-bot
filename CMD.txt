### Moderation
- `$antinuke [enable|disable]` - Enables or disables the anti-nuke module.
- `$automod enable` - Enables the automod system.
- `$automod disable` - Disables the automod system.
- `$automod punishment` - Sets the punishment for automod infractions.
- `$automod ignore channel <channel>` - Excludes a channel from automod.
- `$automod ignore role <role>` - Excludes a role from automod.
- `$automod ignore show` - Displays the channels and roles excluded from automod.
- `$automod ignore reset` - Clears the list of excluded channels and roles.
- `$automod unignore channel <channel>` - Re-includes a channel in automod.
- `$automod unignore role <role>` - Re-includes a role in automod.
- `$automod config` - Shows the current automod configuration.
- `$automod logging <channel>` - Sets the channel for automod logs.
- `$blacklist add <user>` - Blacklists a user from using the bot.
- `$blacklist remove <user>` - Removes a user from the blacklist.
- `$blacklist show` - Shows the blacklisted users.
- `$media setup <channel>` - Configures a channel to allow only media files.
- `$media remove` - Disables the media-only channel configuration.
- `$media config` - Displays the current media-only channel.
- `$media bypass add <user>` - Allows a user to send messages in a media-only channel.
- `$media bypass remove <user>` - Revokes a user's ability to send messages in a media-only channel.
- `$media bypass show` - Lists all users who can bypass the media-only channel restriction.
- `$vcrole add <role>` - Assigns a role to users when they join a voice channel.
- `$vcrole remove <role>` - Removes the role assigned to users in a voice channel.
- `$vcrole config` - Shows the currently configured voice channel role.
- `$whitelist <user>` - Whitelists a user, exempting them from anti-nuke measures.
- `$unwhitelist <user>` - Removes a user from the whitelist, re-enabling anti-nuke measures for them.

### Automation
- `$ar add <trigger> <response>` - Adds an auto-response.
- `$ar remove <trigger>` - Removes an auto-response.
- `$ar list` - Lists all auto-responses.
- `$ar reset` - Resets all auto-responses.
- `$autoreact add <trigger> <emojis>` - Adds an auto-reaction.
- `$autoreact remove <trigger>` - Removes an auto-reaction.
- `$autoreact list` - Lists all auto-reactions.
- `$autoreact reset` - Resets all auto-reactions.
- `$autorole add <role>` - Adds a role to be given to new members.
- `$autorole remove <role>` - Removes a role from the autorole list.
- `$autorole list` - Lists all autoroles.
- `$autorole reset` - Resets the autorole configuration.
